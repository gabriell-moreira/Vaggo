# Vaggo 🚀

Vaggo é uma API REST para gerenciamento de vagas de emprego desenvolvida com Java e Spring Boot, aplicando os princípios da Clean Architecture para garantir escalabilidade, desacoplamento e facilidade de manutenção.

O projeto tem como objetivo servir como estudo prático de arquitetura de software, implementando conceitos como Casos de Uso (Use Cases), Entidades de Domínio, Inversão de Dependência e Separação de Responsabilidades.

## Tecnologias

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Docker
- Clean Architecture

## Funcionalidades

- Cadastro de vagas
- Consulta de vagas
- Atualização de vagas
- Exclusão de vagas
- Filtros e buscas
- Documentação da API

## Arquitetura

O projeto segue os princípios da Clean Architecture, organizando o código em camadas independentes:

- Domain
- Application
- Infrastructure
- Presentation

## Objetivo

Mais do que uma API de vagas, o Vaggo é um projeto focado em boas práticas de desenvolvimento backend, arquitetura de software e construção de aplicações escaláveis utilizando Java e Spring Boot.
